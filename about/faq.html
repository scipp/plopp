Frequently Asked Questions#

Subplots with 3D scatter plots#

The three-dimensional visualizations in Plopp do not use Matplotlib, but Pythreejs. Consequently, it is not possible to insert a three-dimensional scatter plot inside a subplot made of Matplotlib axes. The only way to make a three-dimensional view part of subplots is to use the container widgets described in the subplots page.

[1]:

%matplotlib widget
import plopp as pp
import matplotlib.pyplot as plt
from plopp.widgets import Box

da1 = pp.data.data1d()
da2 = pp.data.scatter()

p1 = pp.plot(da1)
p2 = pp.scatter3d(da2, x='x', y='y', z='z', figsize=(550, 400))

Box([p1, p2])
